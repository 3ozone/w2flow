# Casos de uso

## Flujo descarga licitaciones

## Actores

1. - Web administracion publica => Muestra las nuevas licitaciones si las hay.
2. - Aplicacion => Ejecuta el proceso de descarga aplicando los filtros y criterios para seleccionar las documentacion y descargarla.
3. - Usuario => Revisa las licitaciones descargadas del dia.
